# Kyoto vs Osaka · 京都と大阪の旅のしおり

京都と大阪、どっちに行く？ 見どころを写真つきでまとめた「旅のしおり」ページです。
HTML / CSS / JavaScript だけで作られた、フロントエンドのみのシンプルなページです。

## 見どころ
- 京都7スポット（清水寺・伏見稲荷大社・金閣寺・嵐山・祇園・天龍寺・抹茶）
- 大阪6スポット（大阪城・道頓堀・通天閣・海遊館・梅田スカイビル・たこ焼き）
- スクロールでふわっと表示される演出
- 「どっちに決めた？」の旅先スタンプ機能
